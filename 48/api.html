class demes.Epoch(start_time: Union[int, float] = None, end_time: Union[int, float] = None, initial_size: Union[int, float] = None, final_size: Union[int, float] = None, size_function: str = None, selfing_rate: float = None, cloning_rate: float = None)[source]¶

Population size parameters for a deme in a specified time period. Times follow the forwards-in-time convention (time values increase from the present towards the past). The start time of the epoch is the more ancient time, and the end time is more recent, so that the start time must be greater than the end time

Variables

  • start_time – The start time of the epoch.

  • end_time – The end time of the epoch (must be specified).

  • initial_size – Population size at start_time.

  • final_size – Population size at end_time. If initial_size != final_size, the population size changes monotonically between the start and end times.

  • size_function – The size change function. Common options are constant, exponential, or linear, though any string is valid. Warning: downstream simulators might not understand the size_function provided.

  • selfing_rate – An optional selfing rate for this epoch.

  • cloning_rate – An optional cloning rate for this epoch.

property time_span¶

The time span of the epoch.

class demes.Migration(source: str, dest: str, start_time: Union[int, float], end_time: Union[int, float], rate: float)[source]¶

Parameters for continuous migration from one deme to another. Source and destination demes follow the forwards-in-time convention, of migrations born in the source deme having children in the dest deme.

Variables

  • source – The source deme.

  • dest – The destination deme.

  • start_time – The time at which the migration rate becomes activate.

  • end_time – The time at which the migration rate is deactivated.

  • rate – The rate of migration. Set to zero to disable migrations after the given time.

A collection of individuals that are exchangeable at any fixed time.

Variables

  • description – An optional description of the deme.

  • selfing_rate – An optional selfing rate for this deme.

  • cloning_rate – An optional cloning rate for this deme.

class demes.DemeGraph(description: str, time_units: str, generation_time: Union[int, float] = None, default_Ne: Union[int, float] = None, doi: str = None, demes: List[demes.demes.Deme] = NOTHING, migrations: List[demes.demes.Migration] = NOTHING, pulses: List[demes.demes.Pulse] = NOTHING, splits: List[demes.demes.Split] = NOTHING, branches: List[demes.demes.Branch] = NOTHING, mergers: List[demes.demes.Merge] = NOTHING, admixtures: List[demes.demes.Admix] = NOTHING, selfing_rate: float = None, cloning_rate: float = None)[source]¶

A directed graph that describes a demography. Vertices are demes and edges correspond to ancestor/descendent relations. Edges are directed from ancestors to descendants.

Variables

  • description – A human readable description of the demography.

  • time_units – The units of time used for the demography. This is commonly years or generations, but can be any string. This field is intended to be useful for documenting a demography, but the actual value provided here should not be relied upon.

  • generation_time – The generation time of demes, in units given by the time_units parameter. Concretely, dividing all times by generation_time will convert the deme graph to have time units in generations. If generation_time is None, the units are assumed to be in generations already. See also: in_generations().

  • default_Ne – The default population size to use when creating new demes with deme(). May be None.

  • doi – If the deme graph describes a published demography, the DOI should be be given here. May be None.

  • demes (list of Deme) – A list of demes in the demography. Not intended to be passed when the deme graph is instantiated. Use deme() instead.

  • migrations (list of Migration) – A list of continuous migrations for the demography. Not intended to be passed when the deme graph is instantiated. Use migration() or symmetric_migration() instead.

  • pulses – A list of migration pulses for the demography. Not intended to be passed when the deme graph is instantiated. Use pulse() instead.

deme(id, description=None, ancestors=None, proportions=None, start_time=None, end_time=None, initial_size=None, final_size=None, epochs=None, selfing_rate=None, cloning_rate=None)[source]¶

Add a deme to the graph.

Parameters

  • id (str) – A string identifier for the deme.

  • ancestors (list) – A list of ancestors of this deme. May be None. If len(ancestors) > 1, must also give proportions.

  • proportions (list) – A list of proportions of ancestory for ancestors. Proportions must sum to 1.

  • start_time – The time at which this deme begins existing.

  • end_time – The time at which this deme stops existing. If the deme has an ancestor the end_time will be set to the ancestor’s start_time.

  • initial_size – The initial population size of the deme. If None, this is taken from the deme graph’s default_Ne field.

  • final_size – The final population size of the deme. If None, the deme has a constant initial_size population size.

  • epochs – Additional epochs that define population size changes for the deme.

Variables

  • selfing_rate – An optional selfing rate for this deme.

  • cloning_rate – An optional cloning rate for this deme.

get_demographic_events()[source]¶

Loop through successors/predecessors to add splits, branches, mergers, and admixtures to the deme graph. If a deme has more than one predecessor, then it is a merger or an admixture event, which we differentiate by end and start times of those demes. If a deme has a single predecessor, we check whether it is a branch (start time != predecessor’s end time), or split.

This is only used when we build a demography from a YAML file, since it uses the successors/predecessors that are determined by ancestor relationships.

migration(source, dest, rate=0, start_time=None, end_time=None)[source]¶

Add continuous migration from one deme to another. Source and destination demes follow the forwards-in-time convention, so that the migration rate refers to the movement of individuals from the source deme to the dest deme.

Parameters

  • source – The source deme.

  • dest – The destination deme.

  • rate – The rate of migration per generation.

  • start_time – The time at which the migration rate is enabled. If None, the start time is defined by the earliest time at which the demes coexist.

  • end_time – The time at which the migration rate is disabled. If None, the end time is defined by the latest time at which the demes coexist.

property predecessors¶

Lists of predecessors (ancestors) for all demes in the graph.
