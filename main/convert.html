Conversion functions¶

Warning

These conversion functions are for testing only and it’s expected that this functionality will eventually be integrated into the relevant third-party library.

demes.convert.to_msprime(graph: demes.demes.Graph)[source]¶

Convert a demes graph to an msprime demography.

Parameters

graph (demes.Graph) – the demes graph to convert.

Returns

a 3-tuple (pc, de, mm), where pc is a list of population configurations, de is a list of demographic events, and mm is the initial migration matrix.

Return type

( list of msprime.PopulationConfiguration, list of msprime.DemographicEvent, list of list of float)

demes.convert.from_msprime(population_configurations=None, demographic_events=None, migration_matrix=None, pop_names=None) → demes.demes.Graph[source]¶

Convert an msprime demography into a demes graph.

Parameters

  • population_configurations (list of msprime.PopulationConfiguration) – A list of population configurations.

  • demographic_events (list of msprime.DemographicEvent) – A list of demographic events.

  • migration_matrix (list of list of float) – The initial migration matrix.

  • pop_names – A list of population names to use. If None, the names will be pop0, pop1, …, popN.

  • pop_names – list of str

Returns

A demes graph.

Return type

demes.Graph
